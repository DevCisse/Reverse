                        function reverseString(userString)
                        {
                        
                          let revString =  [];
                          for (let index = userString.length -1 ; index >=0 ; index--) {
                            revString += userString[index];
                            
                          }
                          return revString;
                        }

                        }

The App uses three functions.

getValues()

This function gets inputs from the user. It also adds invisible class to element with id of alert to make the alert invisible. It then calls the reverseString and deisplayString.

reveserString(userString)

This method returns the reverse string of the user input. It uses for loop to iterate through every element in the string and return a reverse.

displayString()

The displayString() method manipulates the DOM by setting the innerHTML of element with id 'msg' to the reversed string and removes the invisible class added previously in the getValues() function.
